## Adam's Laravel Helpers

<p>originally modelled after calebporzio/awesome-helpers</p>

#### Installation:

composer.json
```
"require": {
  "iadambrown/laravel-helpers": "dev-master"
},

...

"repositories": [
  {
    "type": "vcs",
    "url": "https://github.com/iadambrown/laravel-helpers.git"
  }
],
```
